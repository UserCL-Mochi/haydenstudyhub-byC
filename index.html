Imbuhan Mastery 马语词缀

Prefix meN- (主动态)

Function: Active Actions (动作)

Tip: 用于表达主动态动作/行为

Baca → Membaca

Tulis → Menulis

Prefix beR- (自动态/拥有)

Function: State / Condition (状态)

Tip: 表示状态、拥有或自动作

Main → Bermain

Jalan → Berjalan

Prefix teR- (无意/最)

Function: Accidental / Most (无意/最)

Tip: 用于表达无意间发生或最高级

Makan → Termakan

Pandai → Terpandai

Prefix peN- (人物/工具)

Function: Person / Tool (人物/工具)

Tip: 将动词/名词转化为执行者或工具

Lari → Pelari

Sapu → Penyapu

Prefix di- (被动态)

Function: Passive Actions (被动语态)

Tip: 用于第三人称被动句，表示“被”

Makan → Dimakan

Beli → Dibeli

Prefix ke- (序数/名词)

Function: Ordinal / Abstract (序数/抽象)

Tip: 用于数字表示顺序或形成抽象名词

Dua → Kedua

Kasih → Kekasih

Prefix se- (一/同)

Function: One / Same (一/相同)

Tip: 表示“一”、“一样”或“同级别”

Kelas → Sekelas

Tinggi → Setinggi

Prefix peR- (自动态名词)

Function: Result / Abstract (结果/抽象)

Tip: 与beR-对应，形成相关抽象名词

Temu → Pertemuan

Mula → Permulaan
